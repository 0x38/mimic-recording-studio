This folder will hold a subfolder for each recording sesssion.  Each
recording is placed under that subfolder.  The filename of is the hash of
the prompt that was spoken.

Example:

voices / Steve
                ce2d0b870745042c-1526026409.zip   First recording (WAV insde the zip)
                31a1f310512dd949-1526029313.zip   Second recording
       / Michael
                ce2d0b870745042c-1526035412.zip   First recording (WAV insde the zip)
                31a1f310512dd949-1526036323.zip   Second recording

                
The text spoken in each recording can be found in the ../prompts.txt file,
with the same hash value found in the first column of the CSV.
